Landing Page

Una landing page diseñada para promocionar y vender propiedades inmobiliarias, ofreciendo una experiencia de usuario intuitiva y visualmente atractiva.

Tecnologías Utilizadas
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Control de Versiones: Git, Github
  • Despliegue: Netlify
Características Principales
  • Diseño responsivo y atractivo utilizando Bootstrap.
  • Formulario de contacto para consultas de clientes.
  • Mapa interactivo para mostrar la ubicación de las propiedades.
  • Animaciones y efectos visuales con JavaScript para mejorar la experiencia del usuario.
